# Lightweight Intelligence Templates (LIT) SDLC Framework

Provides a simple framework for greatly increasing your chances of success leveraging Artifical Intelligence in your SDLC.  Based on [agent-os](https://github.com/buildermethods/agent-os) v3, it adds customizations to improve memory between sessions, architectural alignment, guardrails, and strategic planning.

## Install

A monorepo will give you the best results.  To install this into your project:

1. Install [Claude Code](https://claude.com/product/claude-code) or [Cursor](https://cursor.com/)
2. Install [agent-os](https://buildermethods.com/agent-os/installation)
3. Drop the files of this project into your repository (with the exception of this README.md)
4. Customize AGENTS.md to provide important context for your project.  These areas are marked with `// TODO:` in the document.
5. Leverage [agent-os](https://buildermethods.com/agent-os) skills as necessary to populate important product context and specs
6. Communicate naturally to build whatever you have scoped.
